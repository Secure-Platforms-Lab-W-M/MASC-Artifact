What is the goal behind MASC?

The goal of the MASC framework is to help refine crytpo misuse detectors to make them as effective as possible. It accomplishes this task by "enabl[ing] a systematic and data-driven evaluation of crypto-detectors using mutation testing." MASC is designed to create mutations in programs that can then be used to test the crypto dectors. This allows the detectors to see which misues cases they may not be covering. This will reduce the flaws seen in crypto-dectors and allow people to be able to trust that these crypto dectors are secure.
