Tipos de combustível

O aplicativo faz distinção entre os diferentes tipos de combustível para compor o relatório de preços, como por exemplo, gasolina ou álcool.

Além disso os tipos de combustível são classificados, como por exemplo gasolina, álcool, diesel ou gás. Se você possui um carro bicombustível (gasolina e álcool, por exemplo), o consumo de combustível é calculado para cada categoria e não para o carro inteiro.
