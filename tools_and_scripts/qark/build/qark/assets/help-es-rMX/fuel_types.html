Tipos de Combustible

La aplicación distingue entre diferentes tipos de combustible para el reporte de precio de combustible. Por ejemplo, gasolina común o gasolina con 10% etanol (E10).

Los tipos de combustible también son categorizados. Por ejemplo, gasolina, diesel o gas. Si tienes un vehículo híbrido, el rendimiento de combustible se calcula por cada categoría y no por vehículo.
