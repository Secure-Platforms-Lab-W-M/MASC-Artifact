General

The whole database gets exported with one file for each table. This means the data of one refueling is separated into multiple files. You can find the car by looking up the value of the car column in the Id column of the cars table.

Columns

cars

_id: [int]
car__name: [string]
color: [int] Have a look at the android color documentation for a more detailed explanation how the value is calculated.
initial_mileage: [int] Initial mileage of the car, when it starts to be used in the app.
suspended_since: [date]

fuel_types

_id: [int]
fuel_type__name: [string]
category: [string]

other_costs

_id: [int]
car_id: [int] the Id of the car.
mileage: [int] Total distance (odometer).
price: [float]
recurrence_interval: [int] 0 = once, 1 = daily, 2 = monthly, 3 = every quarter, 4 = every year
recurrence_multiplier: [int] Specifies the recurrence cycle together with recurrence_interval. Example: interval = 2 and multiplier = 5 means every 5 months.
title: [string]
date: [date]
note: [string]
end_date: [date] End recurrence on this date.

refuelings

_id: [int]
partial: [bool] (1 or 0)
date: [date]
note: [string]
volume: [float]
fuel_type: [int] the Id of the fuel type.
price: [float]
mileage: [int] Total distance (odometer).
car_id: [int] the Id of the car.

reminders

_id: [int]
title: [string]
after_time_span_unit: [int] 0 = days, 1 = months, 2 = years
after_time_span_count: [int] Specifies a time span together with after_time_span_unit. Example: unit = 1 and count = 5 means after 5 months.
after_distance: [int]
start_date: [date]
start_mileage: [int]
car_id: [int] the Id of the car.
notification_dismissed: [bool] (1 or 0)
snoozed_until: [date]
