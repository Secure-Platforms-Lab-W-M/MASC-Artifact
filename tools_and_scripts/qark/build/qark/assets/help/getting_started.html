1. Set up car

Go to cars in the preferences screen to set up your cars and assign them names and colors.

2. Add refuelings and other costs

Get out the app after refueling and quickly enter the costs.
Tip: Ask for the receipt, so you don't need to remember volume and price.

3. View report

After the second refueling you can see the first line in the fuel price report and after the third refueling in the fuel consumption report. The costs report gets more accurate, the more you add in the app.
Tip: Recurring costs, such as taxes must be entered only once and labeled with the appropriate interval. These will then be automatically calculated correctly.
