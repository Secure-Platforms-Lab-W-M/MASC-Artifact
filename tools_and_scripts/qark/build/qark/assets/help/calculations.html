Consumption

To calculate the consumption, the miles are added together by a full tank of fuel to another. For this reason partial refills will only be considered with the next complete fill.

Balance missing refuelings

To recognize missing refuelings the app looks for fuel consumptions, which are much lower (< 80%) than the average. It will then, based on the average distance between refuelings, guess when the missing refuelings probably took place add them temporarily (trying to produce average values for fuel consumption and price) to calculate more balanced values.

These balanced values are used in the fuel consumption report, mileage report, costs report and calculator. The data list displays hints for probably refuelings, but does show the actual values for fuel consumption.
